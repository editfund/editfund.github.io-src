# 教材-代码标注图

#### 简介

draw.io的形状里显示代码，并实现如下特点：

- 代码语法高亮
- 代码可以编辑
- 代码可以复制到VSCode
- drawio可以另建图层（用于标注代码）
- 标注层，用箭头等关联代码片段。



#### 好处

* 明晰代码结构，有助于程序员理清思路。
* 代码复用。

#### 教材

- 把VSCode的背景色设置为drawio色板里有的颜色值

- 从VSCode里复制代码到drawio的矩形
- 为美观，也可以更改该矩形的颜色
