#### 简介

在地图的上层，增加图论层。可以表达一些有地理位置意义的节点与路线。

特点：帮您生成源代码 <a href="https://leafletjs.com/">Leaflet</a> ，基于 <a href="https://leafletjs.com/">leafletjs</a> 的地图展示系统

用途：

* 可以帮助您展示公司、子公司
* 可以帮助您展示水电站与自然资源的地图分布
* 可以帮助您展示物流，原材料运输
