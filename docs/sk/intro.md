# 简介

在 `程序A` 上悬浮工具条程序SK。SK能够输入A，能够给A发送快捷键命令。
- 支持触摸屏
- 常规热键，需要按很多按钮，而且容易忘记，现在有了SK程序，只需按一个按钮
- 段落输入
